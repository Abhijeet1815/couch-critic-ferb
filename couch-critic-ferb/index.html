About Couch-Critic-Ferb

A smart movie recommender built with love and ML.

NLP-Powered Recommendations

Uses Natural Language Processing on movie metadata — genres, keywords, cast, crew, and overview — to compute cosine similarity between movies.

FastAPI Backend

The recommendation engine is served through a high-performance FastAPI backend, supporting async HTTP calls and a clean REST API.

TMDB Dataset

Trained on the TMDB 5000 Movies dataset, containing rich metadata for nearly 5,000 films sourced from The Movie Database.

Scikit-Learn

The ML pipeline uses scikit-learn's TF-IDF vectorizer and cosine similarity to build a content-based filtering model.

TMDB API

Movie posters and metadata are fetched live from the TMDB API, giving users up-to-date visuals for all recommendations.

Open Source

Built as a learning project to combine ML, REST APIs, and modern web design. Fully open-source and hackable.
